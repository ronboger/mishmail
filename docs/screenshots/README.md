# Screenshots

README images live here. They're captured from the built-in **demo inbox**
(fictional mail — no real account), so they're safe to publish.

To (re)generate:

```sh
make demo    # launches PerfectMail Debug seeded with fake mail
```

Then capture the window (**⌘⇧4** then **Space**) and save as:

- `inbox.png`   — the main three-pane inbox (referenced at the top of the root README)

Add more as needed and reference them from the root `README.md`. The fixtures
are defined in `Sources/PerfectMail/Support/DemoSeed.swift`.
