# Screenshots

README images live here. Capture them from the real app (`make run` or
`make install`) — redact/crop anything sensitive (real names, subjects,
addresses) before saving, since these get published.

To (re)generate:

Capture the window (**⌘⇧4** then **Space**) and save as:

- `inbox.png`   — the main three-pane inbox (referenced at the top of the root README)

Add more as needed and reference them from the root `README.md`.
